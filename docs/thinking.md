# 教培AI配图工具 - 思考记录

## 2026-07-03

### 今天做了什么

- 从0搭建React项目
- 接入Pollinations.ai免费文生图API
- 实现三场景选择、输入生成、历史记录
- 部署到Netlify：<https://glowing-mermaid-a329a6.netlify.app/>

### 决策

1. **为什么用Pollinations？**
   - SiliconFlow余额为零，通义万相配置复杂
   - Pollinations免费、免Key、无跨域问题
   - 缺点：速度慢、质量一般，后期可替换
2. **为什么先不做复杂功能？**
   - 原版有素材上传、阶段选择、风格切换
   - 但那些需要后端支持，MVP阶段先不做
   - 核心验证：教培场景+AI生图，这个需求是否成立
   - 先做一个可以用、上线的简略版本，其他功能慢慢来
3. 为什么强调使用组件
   - 避免app.tsx臃肿
   - ai生成代码效果更好，不会影响其他样式

### 问题

- Vercel CLI中文用户名报错 → 改用Netlify Drop拖拽部署
- Tailwind配置问题 → 通常ai给v3版本 自己手动是 v4版本
- Trae有时生成代码不完整 → 关键逻辑手动替换
